--compare--
This script will create a slow.pics link for you that compares two mkv files. The comparison will not appears on the slow.pics homepage.
Source: https://github.com/McBaws/comp/

1. Place two mkv files in this folder.
2. Rename your files:

Simple naming:
av1.mkv
source.mkv

or:

Full naming with fake group names:
[JPBD] Jujutsu Kaisen - S02E01.mkv
[AV1] Jujutsu Kaisen - S02E01.mkv

3. Double-click compare.bat

--lossless-intermediary--
Creates a lossless x265 mkv file. Use this if you have a source mkv that's causing issues.
Warning: x265 lossless files are huge, it's sugggested you only create one at a time.
